Some examples are given on how SOS1 sets and SOS2 sets can arise.
Example 1. Depot Siting
A depot can be sited at any one of the positions A, B, C, D, or E. Only one depot can be built.
If 01 indicator variables di are used to perform the following purpose: di = 1 if and only if the depot is sited at i (i is A, B, C, D, or E), then the set of variables (d1, d2, d3, d4, d5) can be regarded as an SOS1 set.

Although the most common application of SOS1 sets is to modelling what would otherwise be 01 integer variables with a constraint such as (4), there are other applications.
