In view of the (convex) shape of the graph we cannot obtain values for the li which give us points below the piecewise straight line. Therefore we must always obtain corresponding values of x1 and y which lie on one of the line segments by virtue of optimality. Stipulation (4) is therefore guaranteed in this case. We can therefore solve our transformed model by linear programming and obtain a satisfactory optimal solution. It is not even necessary to resort to the separable extension of the simplex algorithm which is discussed below. This happens, however, only because our problem is convex.
For a non-convex problem stipulation (4) would generally not be satisfied automatically. In order to guarantee that it be satisfied we could resort to the separable programming modification of the simplex algorithm. In order to demonstrate the difficulty that a non-convex problem presents we will make a piecewise linear approximation to the non-linear term in the non-convex model of Example 2 in Section 7.2.
