While the methods used are mathematically less sophisticated, it has representational advantages which can make problems easier to model and sometimes easier to solve in view of the more concise representation. Therefore we briefly discuss the subject here. It seems likely that hybrid systems will eventually emerge in which the modelling capabilities of CLP are used in modelling languages for systems which use either CLP or IP for solving the models.
In CLP each variable has a finite domain of possible values. Constraints connect (or restrict) the possible combinations of values which the variables can take. These constraints are of a richer variety than the linear constraints of IP (although these are also included) and are usually expressed in the form of predicates. For example, a very common predicate is all_different. When applied to a set of variables as, for example,
all_different (x1, x2, , xn)
it means that all these variables must take distinct values. While this condition can be imposed using integer programming, it is more cumbersome. Once one of the variables has been set to one of the values in its domain (either temporarily or permanently), this predicate implies that this value must be taken out of the domain of all the other variables (a process known as constraint propagation). In this way constraint propagation is used progressively to restrict the domains of the variables until a feasible set of values can be found for each variable or it can be shown that none exists.
CLP is mainly useful where a problem has no objective function or all that is sought is a feasible solution. It lacks the ability to prove optimality (apart from by complete enumeration) in the way which IP does by using the dual values of an LP relaxation.
There is considerable interest in using predicates such as those found in CLP for modelling in IP or CLP. An early paper is McKinnon and Williams (1989), and Fourer (1998) also discusses this. Comparisons and connections between IP and CLP are discussed by Barth (1995), Bockmayr and Kasper (1998), Brailsford et al. (1996), Proll and Smith (1998), Darby-Dowman and Little (1998), Hooker (1998) and Wilson and Williams (1998). The formulation of the all_different predicate using IP is discussed by Williams and Yan (1999).
